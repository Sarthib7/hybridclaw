# Templates

Files in this directory are runtime bootstrap templates copied into HybridClaw
agent workspaces by `src/workspace.ts`.

They are product inputs, not repo contributor onboarding docs.

Use these locations instead for contributor-facing guidance:

- `README.md` for end users
- `CONTRIBUTING.md` for human contributors
- `AGENTS.md` for repo-level coding agent instructions
- `docs/development/` for deeper maintainer and runtime reference docs
